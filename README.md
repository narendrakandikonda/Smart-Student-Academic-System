# Smart Student Academic & Attendance Management System

## Project Description
A console-based Python application to manage student academic records and attendance using MongoDB.

## Features
- Add student details with alphanumeric roll numbers
- Store multiple subject marks per student
- Calculate total, percentage, and grade
- Attendance management with undo using stack
- Student relationships using graph
- Academic hierarchy using tree

## Technologies Used
- Python
- MongoDB (MongoDB Compass)
- Data Structures (List, Stack, Tree, Graph)

## How to Run
1. Start MongoDB server:
   mongod --dbpath C:\data\db

2. Run the project:
   python main.py

## Note
MongoDB must be installed and running before executing the program.
